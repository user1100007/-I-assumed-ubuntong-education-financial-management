## Getting Started

### Logging In
1. Open the EFMS application in your web browser
2. Enter your username and password
3. Click **Sign In**
4. You will be directed to the Dashboard

### Language Selection
- Click the language link in the top-right corner
- Choose **English** or **ខ្មែរ** (Khmer)
- The interface will update immediately

### Selecting Fiscal Year
- The fiscal year (e.g., "Fiscal Year 2025") is displayed at the top of the page
- To change fiscal year, use the **Settings** menu (gear icon → Fiscal Year)

## Budget Planning

### Annual Budget Planning Workflow

#### Step 1: Prepare
1. Navigate to **Budget Plan Preparation** → **Annual Budget Planning**
2. Review the school information at the top
3. Verify fiscal year is correct

#### Step 2: Input Budget Data
1. Enter budget amounts for different categories
2. Use the currency format with comma separators (e.g., 1,000,000)
3. The system will calculate sub-totals automatically
4. For multi-level schools, review both main and sub-level sections

#### Step 3: Validate
- Ensure all required fields are filled
- Check that budget totals match expectations
- Review any error messages and correct them

#### Step 4: Save
1. Click **Save** button
2. Wait for confirmation message
3. If successful, you will see "Data is successfully saved"

#### Step 5: Export
1. After saving, you can export the budget plan
2. Use **Download** or **Export** buttons to save as Excel/PDF

### Performance Indicators
1. Go to **Performance Indicator and Target**
2. Define key performance indicators (KPIs)
3. Set target values for each KPI
4. Track progress during the fiscal year

## Budget Execution

### Upload Excel System
1. Navigate to **Budget Execution/Implementation** → **Upload Excel System**
2. Click **Upload Excel**
3. Select your prepared Excel file
4. Review the preview before confirming
5. Click **Submit** to upload
6. Check for validation errors
7. If successful, data is now in the system

### Recording Transactions

#### Receipt Transaction
1. Go to **Receipt Transaction**
2. Click **Add New** or **+**
3. Enter receipt details:
   - Receipt date
   - Source of fund (Government, Donation, etc.)
   - Amount
   - Description
4. Click **Save**

#### Disbursement Transaction
1. Go to **Disbursement Transaction**
2. Click **Add New**
3. Enter disbursement details:
   - Payment date
   - Expense category
   - Amount
   - Description / notes
4. Click **Save**

### Managing Refunds
1. Navigate to **Refund Cash On Hand** or **Cash Payment at Bank**
2. Click **Add New Transaction**
3. Enter refund/payment details
4. Attach supporting documents if required
5. Click **Save**

## Inventory Management

### Uploading Inventory
1. Go to **Material Accounting Registration** → **Upload Inventory**
2. Download the inventory template (if available)
3. Prepare your inventory data in Excel
4. Click **Upload**
5. Select your file and click **Submit**
6. Review any errors and correct them

### Stock Transactions

#### Stock In (Additions)
1. Go to **Stock In Transaction**
2. Click **Add Item**
3. Enter:
   - Item description
   - Quantity
   - Unit price
   - Source (purchase, donation, etc.)
4. Click **Save**

#### Stock Out (Removals)
1. Go to **Stock Out Transaction**
2. Click **Create New**
3. Enter:
   - Item to remove
   - Quantity
   - Reason (use, loss, donation, etc.)
4. Click **Save**

### Inventory Analysis
1. Navigate to **Inventory Analysis**
2. View current inventory status
3. Review quantities and values
4. Check for low-stock items
5. Generate reports if needed

## Financial Reporting

### Generating Reports

#### Annual Budget Plan Report
1. Go to **Cash Book and Statement** → **Annual Budget Plan Report**
2. Select the fiscal year
3. Click **Generate Report**
4. Review the displayed report
5. Click **Export** to download as PDF/Excel

#### Cash Book and Financial Statement
1. Go to **Cash Book and Financial Statement**
2. Select date range (month/quarter/year)
3. Click **View Statement**
4. Review transactions and balances
5. Use **Export** to download

### Report Features
- **Filtering**: Filter by date, category, or status
- **Sorting**: Click column headers to sort
- **Export**: Download in Excel/PDF format
- **Print**: Use browser print function

## Settings & Support

### Account Settings
1. Click your **Profile Icon** (top-right)
2. Select **Setting** → **Account**
3. Options:
   - **Change Display Name**: Update your name
   - **Change Password**: Update your password
   - **Fiscal Year Settings**: Configure fiscal year
   - **School Level Settings**: Configure school levels

### Changing Password
1. Click profile → **Setting** → **Account**
2. Enter:
   - Current password
   - New password (min 8 chars, must include uppercase, lowercase, number, special character)
   - Confirm new password
3. Click **Change Password**

### Fiscal Year Settings
1. Click profile → **Setting** → **Fiscal Year**
2. Select or add fiscal years for your school
3. Click **Save**

### Feedback
1. Click the **green chat button** in the bottom-right
2. Or navigate to **FeedBack** from the menu
3. Enter your feedback/report
4. Click **Submit**

### Support Resources
- **About System**: Information about EFMS
- **Website and Document**: Links to official documentation and external resources

## Troubleshooting

### Common Issues

| Issue | Solution |
|-------|----------|
| **Can't log in** | Verify username/password. Contact administrator. |
| **Excel upload fails** | Ensure file format is correct (.xlsx or .xlsb). Check for required columns. |
| **Khmer text not displaying** | Install Khmer fonts or update browser. Try clearing cache. |
| **Calculations incorrect** | Verify no special characters in numbers. Re-save the form. |
| **Can't save changes** | Check browser permissions. Try a different browser. Clear cache. |

## Tips & Best Practices

✅ **Do:**
- Save your work frequently
- Use the templates provided for uploads
- Review calculated totals before finalizing
- Keep fiscal year selection consistent
- Check both English and Khmer labels for clarity

❌ **Don't:**
- Use commas in currency input (the system adds them)
- Leave required fields blank
- Mix fiscal years in a single operation
- Upload without validating data first
- Share your account credentials
